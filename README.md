Description

### EN
This project contains solutions to problems from a university course: Introduction to Machine Learning and AI.

### RU
Этот проект содержит решения задач по курсу университета: введение в машинное обучение и ИИ.


Tools
-
Scala 3 in Jupyter Notebook via the Almond kernel. The org.jfree:jfreechart:1.5.6 packages are used for graphics output and org.apache.commons:commons-math3:3.6.1 for solving systems of equations. A Docker file is available for standalone execution.

Scala 3 в Jupyter Notebook через Almond kernel. Используются пакеты org.jfree:jfreechart:1.5.6 для вывода графики и org.apache.commons:commons-math3:3.6.1 для решения систем уравнений. Для автономного запуска есть Docker файл.
